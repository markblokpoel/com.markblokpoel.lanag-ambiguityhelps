RSA1ShotInteraction 

case class RSA1ShotInteraction(agent1: RSA1ShotAgent, agent2: RSA1ShotAgent, originData: OriginData = OriginData(), pairId: Long = InteractionIdentifier.nextId, maxTurns: Int = 10) extends Interaction[ReferentialIntention, ContentSignal, RSA1ShotAgent] with Product with Serializable

This implements a 1-shot director matcher simulation between two Rational Speech Act agents. Agents take turns communicating a randomly assigned referent. Entropy of production and interpretation, and successes are measured.

agent1

One of two agents that is part of this interaction, must be a subtype of com.markblokpoel.lanag.core.Agent.

agent2

One of two agents that is part of this interaction, must be a subtype of com.markblokpoel.lanag.core.Agent.

originData

The data specifies the agents origin, must be a subtype of com.markblokpoel.lanag.core.Data. Default value is com.markblokpoel.lanag.core.NoData.

maxTurns

The maximum number of turns, i.e., the number of signals produced, in this interaction.
