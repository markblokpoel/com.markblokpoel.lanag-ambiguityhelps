    Use to generate pairs of agents who's lexicons are binary, consistent (each referent has at least 1 associated word) and have a fixed ambiguity (viz., each word is associated with ambiguity referents).

    Use to generate pairs of agents who's lexicons are binary, consistent (each referent has at least 1 associated word) and have a fixed ambiguity (viz., each word is associated with ambiguity referents). The range of ambiguity goes from no ambiguity to full ambiguity. One agent's lexicon is generated (within set constraints) at random, the other is a randomized derivative adhering to the same consistency constraints.
