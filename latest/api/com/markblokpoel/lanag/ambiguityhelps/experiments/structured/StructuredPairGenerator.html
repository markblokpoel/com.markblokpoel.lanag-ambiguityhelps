StructuredPairGenerator 

class StructuredPairGenerator extends PairGenerator[ParametersStructured, ReferentialIntention, ContentSignal, RSA1ShotAgent, OriginData] with Serializable

Generates, for specific parameters, a pair of agents with random binary lexicons based on structured binary string representations of the vocabulary and context..

Annotations

    vocabularySize

    The size of the agents' vocabularies.

    contextSize

    The size of the agents' contexts.

    representationLength

    The length of the binary representations used.

    mappingFunction

    A function that takes two binary string representations and computes the relationship between them.

    thresholdResolution

    Specifies that the range of thresholds within the parameter space, from 0 to 1 by thresholdResolution. Threshold is used to determine signal-referent relationships, based on mappingFunction. When mappingFunction >= mappingThreshold, there is a 1.0-value relationship or 0.0 otherwise.

    changeResolution

    Specifies how fine-grained the parameter space is. I.e., derived lexicons will be generated for mutations at change = 0 to 1 with step size changeResolution.

    sampleSize

    The number of agent pairs that can be sampled from each point in the parameter space.

    thresholdLowerBound

    The lower bound of the threshold range.

    thresholdUpperBound

    The upper bound of the threshold range.

    changeLowerBound

    The lower bound of the change range.

    changeUpperBound

    The upper bound of the change range.

    beta

    The decision noise parameter used in the softargmax inference of the agents.
