  •  package random

    This package contains all the necessary classes for running a Rational Speech Act simulation where agent pairs are generated using a random and mutation strategy.

  •  package structured

    This package contains all the necessary classes for running a Rational Speech Act simulation where agent pairs are generated using a strategy that generates binary string representations for referents and signals.

  •  package uniform

    This package contains all the necessary classes for running a Rational Speech Act simulation where agent pairs are generated with uniform lexicon ambiguity (i.e., each signal refers to the same number of referents).
