  1.  new RSA1ShotAgent(originalLexicon: Lexicon, order: Int = 0, beta: Double = Double.PositiveInfinity)

    originalLexicon

    The original (initial) lexicon of the agent.

    order

    The default order of the agent (default to 0).

    beta

    The decision noise parameter. Used in the signal and referent inference, which is based on the softargmax function. Defaults to PositiveInfinity, making softargmax equivalent to argmax (i.e., purely rational agents).
